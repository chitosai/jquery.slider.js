### Feature(?)

**Lightweight**   
Only 100+ lines, 1.91kb compressed(ungzipped).    
Super easy to use, super easy to edit.   

### Usage:

HTML:
```
<div class="slider">
  <any-node>anything</any-node>
</div>
```

CSS:
```
.slider-pagi{
  /* style of the navi bar */
}

.slider-control{
  /* style of the navi item */
  &.active{
    /* style of navi item that is currently active */
  }
}
```

Javascript:
```
$('.slider').slider();
```

#### That's all. Enjoy :)
